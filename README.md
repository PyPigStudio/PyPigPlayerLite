# PyPigPlayerLite

PyPigPlayer的精简版本。

# 功能对比

| 功能         | PyPigPlayer v1.0a3.5       | PyPigPlayerLite            |
| ------------ | -------------------------- | -------------------------- |
| 支持文件格式 | MP3、LRC                   | MP3、LRC                   |
| 基础播放功能 | 进度条、快进快退、音量调节 | 进度条、快进快退、音量调节 |
| 循环模式     | 播放一次/单曲循环          | 仅单曲循环                 |
| 滚动歌词     | 自滚动、鼠标滑轮、上下按钮 | 自滚动、鼠标滑轮           |
| 在线搜索     | 仅歌词                     | 仅歌词                     |
| 窗口模式     | 可更改窗口大小，支持全屏   | 固定400\*600，不支持全屏   |
| 窗口背景     | 自定义图片/自定义颜色      | 默认图片                   |
| 占用空间     | 45M                        | 14M                        |
